Student Stress Analysis Project

Overview

This project analyzes stress levels among university undergraduate students using survey data. The study aims to identify the major factors contributing to student stress and provide insights through data analysis.

Objectives

* Collect and analyze student stress data.
* Identify key factors affecting student stress.
* Examine relationships between academic and personal stress factors.
* Provide recommendations to improve student well-being.

Project Files

* Report.pdf – Final project report
* Stress raw data.csv – Raw survey dataset
* Stress survey responses.xlsx – Survey responses
* Survey Link.pdf – Survey questionnaire and survey link

Methodology

1. Data collection through an online survey.
2. Data cleaning and organization.
3. Data analysis of survey responses.
4. Interpretation of findings and report preparation.

Tools Used

* Microsoft Excel
* CSV Datasets
* Online Survey Forms

Conclusion

This project provides insights into the stress levels of university undergraduate students and identifies the key factors that contribute to student stress.
